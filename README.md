# 🚀 Space Shooter

A simple 2D Space Shooter game built with **Godot 4** and **GDScript**.

This is my **first game development project**, created after learning Godot Engine for just **3 days**. The goal of this project was to understand the fundamentals of game development, scene management, and scripting.

## ✨ Features

- 🚀 Smooth player movement
- 🔫 Bullet shooting system
- 👾 Automatic enemy spawning
- 💥 Collision detection
- ⏱️ Timer-based enemy generation
- 🎯 Basic game loop
- 🌌 Pixel-art space environment

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/princekumar-git10/space-shooter-godot.git
```

### Open the project

1. Install **Godot 4**
2. Open Godot
3. Click **Import**
4. Select the `project.godot` file
5. Press **Run ▶️**
